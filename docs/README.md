# KubeRise OpsLab

A hands-on Kubernetes lab project built using Rocky Linux VMs and kubeadm to understand Kubernetes internals, networking, container runtime configuration, and cluster administration.

## Phases

- Phase 1 — Infrastructure & Base System Setup
- Phase 2 — Kubernetes Cluster Initialization

## Environment

- VMware Workstation Pro
- Rocky Linux 9
- containerd
- Kubernetes v1.36.0
- Flannel CNI
